# Gone Astray
### Made in under 48 hours with 3 people team

##### Fun Dungeon Crawler with simple gameplay mechanics.
##### Procedural Map Generate tech demo.
### Team:
* @eren-darici
* @omerakcali
* @DerekDincer

### How to play?
 * WASD to Move
 * Arrow Keys to Shoot
 * Shift to Dash
 * Use the mini arrow at the bottom of the screen to find boss.
